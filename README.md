# 📝 AI Handwriting Generator

A modern AI-inspired handwriting generation web application built using Python Flask.  
This project converts typed digital text into realistic handwritten notebook-style pages using a custom handwriting font.

---

# 🚀 Features

✅ Convert typed text into handwritten notes  
✅ Realistic notebook page design  
✅ Custom handwriting font support  
✅ PDF export support  
✅ Image download support  
✅ Multi-line paragraph generation  
✅ Clean modern UI  
✅ Google copied text support  

---

# 🛠️ Technologies Used

- Python
- Flask
- Pillow (PIL)
- HTML
- CSS
- ReportLab
- Custom TTF Font

---

# 📸 Project Preview

The application generates notebook-style handwritten pages from typed text.

---

# ⚡ How It Works

1. User enters or pastes text  
2. Flask backend processes the text  
3. Pillow renders the text using a custom handwriting font  
4. Notebook-style page is generated  
5. User can download output as image or PDF
